- Seção "sobre mim";
- Exibição das minhas principais tecnologias e habilidades;
- Registros dos meus projetos realizados;
- Links para acessar minhas contas no Linkedin e GitHub.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Vercel para deploy
